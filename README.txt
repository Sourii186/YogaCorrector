AI Yoga Pose Corrector


Prerequisites - 
1. Python 3.x installed on your system.
2. pip (Python package installer) installed.
3. A good webcam
4. A well lit environment


Installations Steps - 
1. Open VSCode and navigate to the project directory.
2. Install the required dependencies using the command 
pip install -r requirements.txt
3. Run the main.py file.
4. Access the webpage by either using the development server or by the below link.
http://127.0.0.1:5000/
